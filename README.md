

# Название проекта

CarParts

## Установка

### Склонируйте репозиторий:

```bash
git clone https://github.com/Lir00f/CarParts.git
```

### Перейдите в каталог проекта:

```bash
cd CarParts
```

### Установите зависимости:

```bash
pip install -r requirements.txt
```

### Примените миграции:

```bash
python manage.py migrate
```

### Запустите сервер:

```bash
python manage.py runserver
```
